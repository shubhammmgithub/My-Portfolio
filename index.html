GoodFoods Noida Agent

This End-to-End AI Reservation Agent is a sophisticated digital concierge designed to automate restaurant discovery and bookings in Noida. Built with a three-tier architecture, it features a Streamlit frontend, a FastAPI backend, and a SQLite database. The "brain" utilizes Gemini 2.5 Flash with Function Calling to interact with real-time data and a custom Memory Layer that persists user preferences across sessions. By bridging the gap between natural language and structured API actions, the agent demonstrates advanced LLM orchestration, tool integration, and state management—essential competencies for a modern AI/ML Engineer.
